# 第16週 程式(正課)
## step1 利用ellipse()畫出圓形,設定圓心,寬,高,再利用void setup()設定size()大小,在draw()有background(RGB三色)
```c
void setup(){//設定,每秒60次
  size(400, 200);
}
void draw(){//畫,每秒60次
  background(23, 94, 32);//RGB
  ellipse(50, 50, 80, 80);
}//      畫圖 圓心 寬  高
```
